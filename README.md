# Finanças CLI (Python)

Aplicação de linha de comando para registrar **receitas** e **despesas**, categorizar, filtrar por período e gerar relatórios (resumo, por categoria e top gastos).

## O que este projeto demonstra
- Condicionais: `if`, `elif`, `else`
- Estruturas de dados: `list`, `dict`
- Funções e organização de código
- Validação de entrada (sem bibliotecas externas)

## Funcionalidades
- Adicionar transações (receita/despesa)
- Listar transações (com filtros)
- Remover transação por ID
- Resumo (saldo, total de receitas e despesas)
- Relatório por categoria
- Top N despesas
- Dados iniciais de exemplo (opcional)

## Como executar
Requer Python 3 (recomendado 3.10+).

```bash
python financas.py
